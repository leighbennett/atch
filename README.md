<p align="center">  Atch.js is a javascript library that manages file selection, previews, compression and uploads. </p>

## Methods: ##

### add_ui() ###
  ***Description*** 
  
  The .add_ui() method creates a html element for user to select files.
  
  ***Syntax***
  
     Atch.add_ui({
         id:'input1',
         types:['jpeg','png','jpg'],
         colors : ['blue','green']
     });

  ***Parameters*** 
  
  **id**
     Unique string 
  **types**
     Array of file types
  **colors**   
     Array of two colors
  
 
  ***Example***
    
    Atch.add_ui({
         id:'input1',
         types:['jpeg','png','jpg'],
         colors : ['blue','green']
     });
